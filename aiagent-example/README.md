# AI Agent Example
AI Agent example built with OpenAI Agents SDK + Node.js

## How to Run

1. git clone
2. cd aiagent-example
3. npm install
4. copy .env.example as .env
5. fill .env variables
6. npm run server
7. add some customers by Postman/Insomnia
8. change the prompt at index.js bottom
9. npm start
