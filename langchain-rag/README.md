# LangChain RAG Example
LangChain + Node.js + OpenAI RAG example.

## How to Run

1. git clone
2. cd langchain-rag
3. npm install
4. copy .env.example as .env
5. fill .env variables
6. put a file.pdf in the same folder
7. npm start
8. ask questions about your PDF
