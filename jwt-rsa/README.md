# JSON Web Token - RSA Example
Node.js webapi example with JWT auth + RSA cryptography.

## How to Run

1. git clone
2. cd jwt-rsa
3. npm install
4. create .env file
5. fill .env variables as .env.example
6. create your keys files and replace the examples
7. npm start
