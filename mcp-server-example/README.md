# MCP Server Example
Node.js + MCP SDK example.

## How to Run

1. git clone
2. cd mcp-server-example
3. npm install
4. npm run build
5. npm start

## How to Test

1. Download Claude Desktop
2. edit claude_desktop_config.json
3. Settings > Developer and paste the new config
4. restart Claude Desktop
5. ask the price abou any crypto pair
